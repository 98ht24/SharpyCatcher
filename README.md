## Commands
| Command  | Usage |
| ------------- | ------------- |
| ``sc!refresh`` / ``sc!reload``  | Reload the Config  |
| ``sc!echo`` / ``sc!run`` / ``sc!shell``  | Let the Catcher's account write something  |
